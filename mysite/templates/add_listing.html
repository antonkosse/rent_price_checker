How to Add a Listing
  1. Visit rieltor.ua or dom.ria.com
  2. Search for rental properties you're interested in
  3. Open the listing page for a specific property
  4. Copy the full URL from your browser's address bar
  5. Paste the URL in the form above and click "Add Listing"
Important Notes
  • Only URLs from rieltor.ua and dom.ria.com are supported
  • Make sure to use the direct property listing URL, not a search results page
  • Properties are checked every 12 hours for changes
